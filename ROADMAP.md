# FinSight AI Roadmap 🗺️

This roadmap outlines the planned features and milestones for FinSight AI.

- [x] Initial React + Node.js Repository Setup
- [x] JWT Authentication & RBAC Security Layer
- [x] MongoDB Integration & Portfolio Data Models
- [x] OpenAI GPT-4 Integration for basic Chat & Analysis
- [x] Real-time Financial Calculations (SIP, EMI, CAGR)
- [x] Dockerization & CI/CD Pipelines

- [ ] Kubernetes Helm Charts deployment
